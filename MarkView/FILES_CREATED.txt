FILE DESCRIPTIONS
==================

APP LAYER
---------
MarkViewApp.swift
  - Main entry point with @main attribute
  - Document-based app for opening .md files
  - Menu bar: File (Open, Open Folder, Save, Export PDF), View (Toggle Theme, TOC, File Tree)
  - Keyboard shortcuts configured
  - Environment object setup (ThemeManager, WorkspaceManager)

MODEL LAYER
-----------
FileNode.swift
  - Represents file/folder in workspace tree
  - Identifiable, ObservableObject, Hashable
  - Recursive tree loading
  - Markdown file detection
  - Sorting: folders first, then alphabetical

DocumentState.swift
  - HeadingItem: Table of contents entry
  - OpenTab: Represents open file with state
  - Includes content, modified flag, headings, scroll position

ThemeManager.swift
  - ObservableObject for theme management
  - @AppStorage integration for persistence
  - Supports: Light, Dark, System themes
  - Notifies observers of theme changes
  - CSS class and color helpers

WorkspaceManager.swift
  - Manages open files and folders
  - Tab system with active tab tracking
  - File watching via DispatchSource
  - Recent files persistence (JSON)
  - Markdown heading extraction
  - File I/O operations

VIEW LAYER
----------
ContentView.swift
  - Main 3-panel layout via NavigationSplitView
  - Left: FileTreeView (togglable)
  - Center: TabBarView + EditorView
  - Right: TOCView (togglable)
  - Toolbar with theme toggle, panel toggles
  - File drop support

FileTreeView.swift
  - Recursive file tree display
  - Search/filter capability
  - Context menu: Show in Finder, Copy Path
  - Single click to open files in tabs
  - Folder disclosure groups

EditorView.swift
  - NSViewRepresentable wrapper for WKWebView
  - Loads index.html from Resources/Editor
  - WKUserContentController setup
  - Bridge message handling
  - Content loading and theme sync
  - PDF export integration

TOCView.swift
  - Right sidebar showing table of contents
  - Indented headings based on level
  - Click to scroll to heading
  - Active heading highlighting
  - Smooth animations

TabBarView.swift
  - Horizontal tab bar for open files
  - Shows file name with icon
  - Modified indicator (orange dot)
  - Close button (x)
  - Click to switch tabs

BRIDGE LAYER
------------
WebViewBridge.swift
  - Central JS-Swift communication hub
  - WKScriptMessageHandler implementation
  - Message routing system
  - JavaScript evaluation wrappers
  - Delegate pattern for event handling
  - Type definitions for bridge messages

PDFExporter.swift
  - PDF export functionality
  - NSSavePanel integration
  - WKPDFConfiguration setup
  - A4 size output
  - Layout preparation/restoration
  - Error handling with user alerts
